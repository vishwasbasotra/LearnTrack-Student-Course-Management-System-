Setup Instructions
============
### Requirement:
1. **JDK Version: 25.0.1**
2. **Install any IDE**
3. Open the Project Named as **"LearnTrack-Student-Course-Management-System"**
4. Navigate to **"LearnTrack-Student-Course-Management-System\LearnTrack\src\com\vishwas\ui\"** and open Main.java
5. Run the current file and start playing with the system.

#### Sample Hello World Program:
```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
